SRMS CONNECT

Project Overview

This project focuses on the upgradation of the college campus. It was developed using Vue.js, Django, and Microsoft Power BI. The main goal was to digitalize every process in college management.

Features:

  • Event Notifier: Real-time alerts for important department events and deadlines.
  • Student Dashboard: Personalized student portal for tracking assignments, grades, and schedules.
  • Am I Clean Module: Interactive tool to assess personal hygiene and health.
